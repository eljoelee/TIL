### 1. 현 프로젝트 내 테라폼 소스 수정
* 로드밸런서 헬스체킹 'unhealthy' 상태 분석
    * 대상 서버들의 8080 포트가 서비스되고 있지 않음
    * 각 서버별 8080 포트 서비스 실행 명령어 확인
        * tomcat : ./startup.sh
        * java : java -jar *.jar
    * 실행 명령어를 인스턴스(aws_instance)별 'user_data'에 스크립트 형태로 삽입 후 배포
        * 배포 완료하였으나 여전히 'unhealthy' 확인 및 분석 진행 중
